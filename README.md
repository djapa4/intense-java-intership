# intense-java-intership
HR platform zadatak za praksu u kompaniji Intense.

Najzahtevniji deo zadatka bio je implementacija many-to-many veze između kandidata i vestina i poslovne logike za dodavanje i uklanjanje vestina. Takodje, zanimljivo je bilo napisati unit testove za service sloj koristeći Mockito i JUnit. Nije radjena implementacija React UI niti testovi za REST API jer ranije nisam koristio te tehnologije.

Takodje, posto sam video da je pozeljno i poznavanje mikro servisa, imam projekat koji je dodat na github-u koji je ukljucivao i sam rad sa mikro servisima na fakultetu.
